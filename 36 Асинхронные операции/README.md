# Асинхронные операции

## Интуитивное понятие асинхронности

Синхронный подход:
1. Поставить чайник
2. Дождаться закипания, выпить чай, начать уборку 

Асинхронный подход:
1. Поставить чайник
2. Начать уборку
3. Когда закипит, выпить чай и продолжить уборку
